# Some really cool cli tools

## exa

> ls replacement, with coloration and speedy

## tokei

> Display statistics about the number of code lines in a given folder

## bat

> Cat replacement, with line count and coloration

## fzf

> Fuzzy search

## fd

> Find replacement

## procs

> See running process list

## sd

> Find and replace

## dust

> Measure size of things on you filesystem

## ripgrep

> Recursive search

## zoxide

> Instant jump to frequently used folders.

Use `zi` to trigger an interactive search.

## navi

> Simple cheat sheet
